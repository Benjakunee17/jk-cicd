# jk-cicd


Terraform-Based CI/CD Pipeline with Jenkins, Maven, Docker, and SonarQube

Description

- Provisioned AWS infrastructure using Terraform including EC2 instances for Jenkins and SonarQube.

- Implemented Jenkins pipelines for Maven build, Docker image creation, and container deployment.

- Integrated SonarQube code quality analysis into the CI pipeline.

- Automated application deployment using Docker containers running on EC2.
